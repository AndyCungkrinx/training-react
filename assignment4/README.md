## How to Install
Only work in node 14.x
```sh
git clone https://github.com/AndyCungkrinx/training-react.git
cd training-react/assignment-3
npm install -f
npm i fsevents@latest -f --save-optional
npx react-codemod rename-unsafe-lifecycles -f
```

## How to Run

```sh
npm run dev
```
