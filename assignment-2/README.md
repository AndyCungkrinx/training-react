# How to install
- git clone https://github.com/AndyCungkrinx/training-react.git
- cd assignment-2
- npm install -f
- npm install --unsafe-perm node-sass
- npm install react-scripts react-redux redux redux-thunk formik redux-devtools-extension yup --save

# How to run
- npm run start
